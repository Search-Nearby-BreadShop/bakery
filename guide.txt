[디자인가이드]
| 항목      | 기준                | 설명                           |
| ------- | ----------------- | ---------------------------- |
| 단위      | rem               | 루트 폰트크기 기준으로 반응형 적용          |
| 기본 폰트크기 | 1rem = 10px       | (html { font-size: 62.5%; }) |
| 지도 높이   | 60vh~70vh         | 뷰포트 기준                       |
| 버튼 크기   | 1.2rem ~ 1.4rem   | 접근성 고려                       |
| 입력 필드   | 반응형 너비 (30~40rem) | rem 단위로 고정 크기 배제             |

[향후확장]
| 구분 | 확장 기능                   |
| -- | ----------------------- |
| 1  | 로그인 기능 (카카오계정 연동)       |
| 2  | 찜 목록 공유 기능 (URL 기반)     |
| 3  | 댓글 좋아요, 신고 기능           |
| 4  | 반경 입력 커스터마이징 (사용자 직접입력) |
| 5  | 지도 스타일 커스터마이징 (다크모드 등)  |
| 6  | 반응형 레이아웃 완성 (모바일 최적화)   |
